# TabletBrowse Pro

专为平板设备优化的浏览器插件，提供更好的网页浏览体验。

## 主要功能

### 🚫 页面滑动行为禁用
- 完全禁用页面的左右滑动行为
- 防止意外页面导航和后退
- 保持稳定的浏览体验

### 📝 字体大小调节
- 50%-200% 字体大小调节范围
- 实时生效，无需刷新页面
- 智能保持按钮和表单元素可用性

### 📜 滚动条增强
- 平板友好的滚动条设计
- 平滑滚动动画效果

## 安装使用

1. 打开Chrome浏览器，进入 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目文件夹

点击浏览器工具栏中的插件图标即可配置：
- 启用/禁用滑动行为禁用
- 调节字体大小
- 控制插件开关

## 技术特性

- **实时设置同步**: 设置立即生效，无需保存按钮
- **触摸事件优化**: 专为平板触摸设计
- **轻量级架构**: 最小化性能影响
- **跨平台兼容**: 支持所有Chromium浏览器
